# Customer Satisfaction Website

A tablet-friendly customer satisfaction survey website built with Bootstrap and Docker.

## Features
- **Responsive Design**: Optimized for tablet touchscreens.
- **Star Rating System**: Intuitive 5-star feedback mechanism.
- **Dockerized**: Ready for deployment using Docker containers.
